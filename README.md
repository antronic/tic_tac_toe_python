###Requirement
----
* Python3.X

###How to use it
----
1) Your computer have to installed Python 3.X<br />
2) Clone this repo
```sh
$ git clone https://github.com/antronic/tic_tac_toe_python.git
```
3) Go to tic_tac_toe_python directory
```sh
$ cd tic_tac_toe_python
```
4) Run it!
```sh
$ python tic_tac_toe.py
```
3) Enjoy it!
